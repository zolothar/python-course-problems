Напишите функцию `assess_yield()`, которая будет вычислять перспективность урожая и давать ему оценку.

У функции `assess_yield()` — два параметра:

- `number_of_plants` — количество растений, целое число;
- `average_weight` — средняя урожайность одного растения, целое число (в граммах).

Функция должна подсчитать общий вес урожая со всех растений в килограммах. Например, если число растений — 2, а урожайность каждого — 500 г, то общий урожай будет равен 1 кг.

Функция `assess_yield()` должна возвращать кортеж из двух элементов:

- общий вес урожая **в килограммах,**
- строку — сообщение с оценкой урожайности.

Сообщения должны быть разными, в зависимости от предполагаемого веса урожая:

- Если общий вес урожая строго больше 100 кг, программа должна вернуть сообщение «Ожидается отличный урожай.».
- Если общий вес урожая строго больше 50 кг, программа должна вернуть сообщение «Ожидается хороший урожай.».
- Если общий вес урожая больше нуля — программа должна вернуть сообщение «Урожай будет так себе.».
- Во всех остальных случаях программа должна вернуть сообщение «Урожая не будет.».
