После состязаний каждому участнику должно приходить сообщение, в котором будет

- время, за которое участник прошёл дистанцию — в формате `ЧЧ:ММ:СС`;
- время, на которое участник отстал от лидера — в формате `Ч:ММ:СС`;
- если участник — лидер, то он получает только сообщение о собственном результате.

Например:

```
Вы пробежали за 10:38:42 с отставанием от лидера на 0:13:18. 
```

Напишите функцию `get_results()`, которая принимает два параметра:

- время, с которым финишировал лидер гонки;
- время, с которым финишировал участник, которому приходит сообщение.

Время передаётся строкой в формате `ЧЧ:ММ:СС`.

Функция `get_results()` должна вычислить разницу между этими значениями и вернуть сообщение:

```
Вы пробежали за ЧЧ:ММ:СС с отставанием от лидера на Ч:ММ:СС. 
```

Если же сообщение отправляется лидеру, оно должно быть таким:

```
Вы пробежали за ЧЧ:ММ:СС и победили! 
```
