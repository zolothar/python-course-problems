Система расчёта ежемесячных платежей такова:

1. Клиент называет сумму кредита и количество месяцев, за которые планирует выплатить кредит.
2. Банк делит сумму кредита на названное количество месяцев и увеличивает ежемесячный платёж на определённый фиксированный процент, известный заранее.
3. Клиент ежемесячно выплачивает часть полученной суммы + проценты.

Напишите функцию `get_monthly_payment()`, которая будет принимать три параметра:

- сумму кредита;
- количество месяцев, за которые клиент планирует выплатить кредит;
- процент банка.

Функция должна вернуть значение ежемесячного платежа по кредиту — ежемесячную часть основной суммы + процент.

Информацию нужно распечатать в следующем виде:

```
Ежемесячный платёж: <сумма> ВтК. 
```

Все расчёты выполняйте с числами типа `decimal`.

Общее количество значимых чисел в результате вычислений должно быть `3`.
